# LIBRERÍAS CSS y JAVASCRIPT

## CSS
 

### [>Polyfill -- Position sticky](https://github.com/philipwalton/polyfill/)

Para empezar me gustaría hacer uso del elemento **STICKY** para mantener ciertos elementos en la página incluso cuando el usuario scrollee. Para esto originalmente encontre [*fixed-sticky*](https://github.com/filamentgroup/fixed-sticky), aunque lamentablemente parece estar desactualizado. Pero encontré un reemplazo.

Polyfill -- Position sticky es una librería con distintos elementos, entre ellos un position-sticky que permite que ciertos elementos queden pegados en la página.

### [>Beautiful CSS box-shadow examples](https://getcssscan.com/css-box-shadow-examples)

Recopilación de distintos estilos para darle sombreado, bordes y colores a boxes dentro del CSS. Hay casi 100 opciones y parece ser tan sencillo como copiar y pegar la información.

### [>Halfmoon](https://github.com/halfmoonui/halfmoon)

Framework de CSS que proporciona variedad de estilos y otras *features* para estilizar y diseñar el frontend de CSS. Se ve algo complicada pero seguro leer un poco la info + el github lo resuelve... espero..

## JAVASCRIPT

### [Inferno](https://github.com/infernojs/inferno)

Parece ser una librería llena de herramientas para crear UIs modernas utilizando Javascript. Podría ser utilizada junto con el CSS, pero parece ser algo complicada. Similar a Halfmoon, creo que es necesario leer detalladamente la página de github para entender cómo funciona exactamente, pero por lo visto se ve poderosa.

Al parecer tienen una aplicación *"Create Inferno App"* que es la forma más sencilla e inicial de utiizar Inferno.
