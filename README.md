# pagina-final-lc

Esta página replica el estilo de blogs personales en internet durante los 2000s: Sencillo, minimalista y algo amateur.
La página tiene un propósito claro: Un archivo de reseñas de cómics, en los que el usuario puede acceder tanto a reseñas de *Issues* individuales como de volúmenes (que coleccionan varios Issues) y runs completos de series. El usuario tiene el menú principal con reseñas recomendadas y también podra, desde este menú, buscar por series y sus colecciones (siempre y cuando estén disponibles). 

La idea es tener variedad de tanto issues como de volumenes y runs completos y a su vez transportar al usuiaro a la época temprana de internet: más descentralizada y personal -- sin preocupaciones ni intereferencia corporativa.
